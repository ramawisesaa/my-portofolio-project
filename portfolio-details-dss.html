Project detail

A fully automated system designed to optimize storage placement for virtual machines (VMs) by analyzing utilization metrics such as IOPS, throughput, and disk usage across multiple storage devices. Developed using Python, Prometheus, Grafana, and Google Apps Script, this solution enables capacity planning, workload balancing, and efficient long-term infrastructure management.
